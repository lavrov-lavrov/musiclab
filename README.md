# App

MusicLab pet project

## Running the app

### Dependencies

Install necessary dependencies:

```bash
npm i
```

### Prepare app

Run script to prepare app for local development:

```bash
npm run prepare
```

### Start app

Run this script to start the app for development:

```bash
npm run dev
```

### Build app

Run script to build the app:

```bash
npm run build
```
